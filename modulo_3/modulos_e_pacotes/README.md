# Módulos e Pacotes em Python

Módulos e pacotes são componentes essenciais em Python para organizar e reutilizar código. Um módulo é um arquivo contendo definições e instruções em Python, enquanto um pacote é uma coleção de módulos organizados em diretórios.

## Criando um Módulo

Um módulo é simplesmente um arquivo `.py` contendo funções, classes e variáveis que podem ser importadas e reutilizadas em outros scripts.
